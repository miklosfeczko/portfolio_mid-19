Movie Browser React

Project Brief

My latest project to browse movies using the TMDb API. As the application grew I implemented more and more functionality into it. I used Material UI for responsive navbar and drawer. The user can also view trailers, cast information, biographies.

Challenges

I had to handle numerous API calls on each route. These routes also required different approach and implementation. I also used Material UI to give the user a better experience on smaller devices.
